Architecture

DroneScan Hardware (warehouse)
    → Drone scans RFID/barcode tags
    → Data uploaded to Azure IoT Hub
    → Azure triggers webhook to 2nth.ai
    → agents.2nth.ai runs inventory analysis skills
    → Results: discrepancies, anomalies, compliance flags
    → Push to WMS via DroneScan API
    → Dashboard at scan.dronescan.co (branded skills hub)

Revenue flow:
    → Customer buys DroneScan hardware + subscription
    → AI analysis uses 2nth.ai tokens
    → DroneScan earns 20% of token revenue from their skills
    → Customer sees unified DroneScan bill
