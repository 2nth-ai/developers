What it does

Unified contacts

Partners, resellers, customers, and leads in one place. No more spreadsheets per product.

Revenue pipeline

Lead → Qualified → Proposal → Negotiation → Contracted → Active. Auto-advances on platform events.

Token billing

Paystack webhooks update deal stages. Token purchases, revenue share, and partner earnings tracked per deal.

GraphQL API

Every CRM action available via API. Developer portal enquiries create leads. Skill usage updates activities.

Deployment

Three ways to run 20crm:

Self-hosted + Cloudflare

Docker Compose on a VPS with Cloudflare edge. Full control, SA data residency. ~$20-40/mo.

Twenty Cloud

Managed hosting at twenty.com. Free tier available. Connect via API and webhooks.

Cloudflare Workers proxy

Edge-cached GraphQL proxy in front of Twenty Cloud. Low latency from 2nth.ai platform.
