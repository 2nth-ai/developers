Architecture

ScanMan WMS (warehouse floor)
    → Barcode scans, stock movements, receiving events
    → ScanMan server aggregates data
    → API webhook to 2nth.ai on key events
    → agents.2nth.ai runs warehouse skills
    → Results: accuracy alerts, shelf life warnings, pick optimisation
    → Push back to ScanMan dashboard + mobile app
    → Branded hub at scan.scanman.co.za

DroneScan integration:
    → DroneScan drones perform aerial stock counts
    → Scan data reconciled against ScanMan WMS records
    → AI flags discrepancies between drone count & WMS
    → Both partners' skills work together automatically
