  How the swarm works:

  1. Deploy — Spin up an ERPNext instance per site (Cloudflare Pages + D1)
  2. Opinionated — Pre-configured for the market:
     • Warehouse: Stock Entry, Putaway Rule, Cycle Count, Pick List
     • Port: Container Register, Damage Log, Yard Grid, Vessel Schedule
     • Farm: Crop Cycle, Livestock Register, Weather Integration, Yield Forecast
  3. Scan — Robots feed scan data into the local ERPNext instance
  4. AI — Cloudflare Workers AI runs analysis at the edge
  5. Swarm — Local instances sync to master ERP via:
     • ERPNext Data Import / Export API
     • SAP RFC / BAPI connectors
     • Oracle REST Data Services
     • Microsoft Dynamics Business Central API
  6. Consolidate — Master ERP has global view:
     • Total inventory across all warehouses
     • Fleet utilisation across all sites
     • Compliance status per jurisdiction
     • Consolidated financials in customer's currency
